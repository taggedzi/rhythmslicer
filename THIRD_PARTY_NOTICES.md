# Third-Party Notices

RhythmSlicer Pro depends on the following major third-party components:

- VLC / libVLC: Required for audio playback. Install VLC from VideoLAN.
- python-vlc: Python bindings for VLC/libVLC.
- Textual: TUI framework used for the interface.
- Mutagen: Audio metadata parsing.

Each dependency is distributed under its own license. Please refer to the
respective project pages for full license texts.
